Quick Start

Default Entry Points

  • js: /src/js/entry.js
  • less: /src/style/entry.less>

Development bundles

Your development bundles are written to 

/dist/<package name>/app.bundle.[js|css]

Production bundles

Your production bundles are written to 

/dist/<package name>/app.bundle-<hash>.[js|css]

Commands 

$ NODE_ENV=dev febs dev --no-dev-server
Builds development bundles
Bundles are un-versioned and un-minified
Source maps are inline
$ NODE_ENV=prod febs prod
Builds production bundles
Bundles are versioned and minified
Source maps are external
$ NODE_ENV=dev febs dev
Run development server with live reloading of assets.
